영화등록
    - 이미지 등록 시 X 클릭
        => 서버 폴더에서 이미지 제거
    - 사진만 첨부한 후 글을 작성 안하는 경우
        => 서버에 사진만 남게 됨

영화수정
    - 영화등록 로직과 동일
    - 이미지 등록 시 X 클릭
        => 이미지 li 만 제거

    - 수정 클릭
        => 기본 이미지 제거
        => 다시 수집된 이미지 정보 삽입

시큐리티 적용

    -영화등록(수정, 삭제)은 admin 가능
    -영화보기 모두 가능 (로그인 없이)
    -리뷰 등록, 수정, 삭제 : 로그인 후
    -프로필 메뉴 : 로그인 후

    - 로그인 POST 작성하지 않음 => Security 알아서 해줌
        1) DB 확인 작업을 할 수 있도록 서비스는 만들기 => UserDetailsService 구현

    - 로그아웃 => Security 알아서 해줌
